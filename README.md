# 🔧 CyberCrip Diagnostic Tool

**Script de diagnóstico rápido para PCs Windows** – criado para a assistência técnica **CyberCrip**. Exibe informações essenciais do sistema, hardware e rede, ajudando a identificar gargalos de desempenho e problemas comuns.

---

## 🎯 Funcionalidades

- Sistema operacional e nome do computador
- Modelo do processador (via `wmic`)
- Memória RAM total e disponível
- Espaço em disco por partição (apenas drives fixos)
- Teste rápido de conexão com a internet (ping para 8.8.8.8)
- Sugestões automáticas com base nos recursos detectados

---

## 🖥️ Requisitos

- Windows (7, 8, 10, 11)
- Python 3.6 ou superior
- Nenhuma biblioteca externa – só o que já vem com o Python
